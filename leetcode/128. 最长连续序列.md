时间复杂度O（n）,空间复杂度O(n)
```python
class Solution:
    def longestConsecutive(self, nums: List[int]) -> int:
        longest = 0
        nums_set = set(nums)
        for a in nums_set:
            if a-1 not in nums_set:
                curlen = 0
                curnum = a
                while curnum in nums_set:
                    curlen += 1
                    curnum = curnum+1
                longest = max(curlen,longest)
        return longest
```
