#小顶堆
```java
//小顶堆
PriorityQueue<Integer> queue = new PriorityQueue<Integer>();
queue.add(1);
queue.add(2);
queue.add(3);
System.out.println(queue.peek());
/
```

#大顶堆
```java
/大顶堆
PriorityQueue<Integer> queue2 = new PriorityQueue<Integer>((o1,o2)->(o2-o1));
queue2.add(1);
queue2.add(2);
queue2.add(3);
System.out.println(queue2.peek());
```
